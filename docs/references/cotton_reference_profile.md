# Cotton Inspection Profile Reference

## Defect pattern
Cotton samples can occasionally trigger compact detector responses even when the part remains acceptable. These cases should be checked against the normal texture profile before escalation.

## Sensor signature
Compact cross-modal inconsistency with weak thermal support can occur when the surface texture varies locally. Treat global descriptor drift as a verification cue, not an automatic reject signal.

## Likely causes
- Benign local texture variation
- Small placement or lighting mismatch during capture
- Minor surface irregularity that does not indicate functional damage

## Inspection checks
- Compare the highlighted region against a known-good cotton reference
- Confirm whether the compact region reflects normal texture variation rather than a true defect
- Escalate only if the region grows, repeats, or shows obvious material damage

## Corrective action
Retain as monitored-normal if the region matches expected texture variation. Review capture alignment or baseline normalization if the same cue appears repeatedly.

## Trust notes
Use this profile when cotton inspections show compact detector activity without strong physical evidence of damage.
