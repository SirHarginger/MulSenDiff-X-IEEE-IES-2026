# Spring Pad Break Reference

## Defect pattern
Spring-pad fractures commonly appear as compact anomalies with strong descriptor drift away from the normal spring profile. The detector often assigns high severity even when thermal support is weak.

## Sensor signature
Watch for compact cross-modal inconsistency near the spring edge or loaded contact area. Compare the dominant region against the expected spring contour and compression geometry.

## Likely root causes
- Fatigue fracture after repeated compression
- Edge damage from misaligned insertion
- Local break after excessive load

## Inspection checks
- Inspect the highlighted region for cracked spring material
- Compare contour continuity with a known-good spring pad
- Verify whether the defect sits near a compression hot spot

## Corrective action
Quarantine the part, review insertion alignment and compression force, and inspect upstream tooling for repeated localized overload.

## Trust notes
This reference is intended for compact spring-pad break detections with elevated global descriptor drift.
